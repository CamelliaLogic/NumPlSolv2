\# NumPlSolv2 - Delphiによる数独解析アプリ



Delphi（Object Pascal）で開発された、カメラ画像認識機能付きの数独（ナンバープレース）解析アプリケーションです。

マルチプラットフォームに対応しており、同一のソースコードでWindowsおよびAndroidの両環境で動作します。



\## 📱 対象OS

\- \*\*Windows\*\* （ノートPC等のカメラに対応）

\- \*\*Android\*\* （スマートフォン・タブレットのカメラに対応）

\## 🛠 主要ファイル構成

このプロジェクトの核心となる主要なソースファイルの説明です。



| ファイル名 | 役割・説明 |

| :--- | :--- |

| \*\*Unit1.pas\*\* | 数独アプリのメインプログラム（UI・制御ロジック群） |

| \*\*MyMtrix.pas\*\* | 数独の盤面や計算を処理する配列処理クラス |

| \*\*MyCNN.pas\*\* | 画像から数字を識別する畳み込みニューラルネットワーク（CNN）クラス |

| \*\*MyDLCommon.pas\*\* | CNNの計算に使用する共通の共通手続き（Procedure）および関数（Function） |

| \*\*MyWeightConst.pas\*\* | CNN用の学習済み重み定数データ（その1） |

| \*\*MyWeightInstance.pas\*\* | CNN用の学習済み重み定数データ（その2） |

\## ⚠️ ビルド時の注意点（アイコンデータについて）

本リポジトリには、各デバイス（Windows / Android）用のアプリ外観アイコンデータ（アートワーク）は含まれておりません。



ソースコードをご自身の環境でビルド・実行される際は、お手数ですが以下の手順でアイコンデータを生成してご利用ください。



1\. Delphiでプロジェクトを開く

2\. 画面上部メニューの \*\*\[プロジェクト] ＞ \[オプション]\*\* を開く

3\. \*\*\[アプリケーション] ＞ \[アイコン]\*\* メニューを選択

4\. \*\*「アートワークジェネレータ」\*\* 等を使用し、必要なアイコンデータを生成して適用してください。
